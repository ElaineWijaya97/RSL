## Setup GitHub Pages

To enable GitHub Pages and make your site accessible:

1. Go to your repository: https://github.com/ElaineWijaya97/RSL
2. Click on **Settings** (top right of the repository)
3. Scroll down to **Pages** in the left sidebar
4. Under **Source**, select:
   - Branch: `main`
   - Folder: `/RSL` (if your files are in the RSL subfolder) OR `/ (root)` (if files are in root)
5. Click **Save**
6. Wait 1-2 minutes for GitHub to build your site
7. Your site will be live at: **https://elainewijaya97.github.io/RSL/**

**Note:** If your files are in a subfolder called `RSL`, make sure to select `/RSL` as the folder. If they're in the root, select `/ (root)`.

## Local Development

To run locally:
```bash
python3 -m http.server 8000
```

Then open http://localhost:8000 in your browser.
